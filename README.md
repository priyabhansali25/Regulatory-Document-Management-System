# Regulatory-Document-Management-System
# Project Overview
A Django-based system to manage regulatory compliance documents with features like upload, search, and admin management.

This project is inspired by real-world regulatory workflows and helps streamline compliance processes.

# Features
- 📂 Upload compliance documents  
- 🔍 Search and filter functionality  
- 🛠 Admin dashboard  
- 📁 Organized document storage  

# Tech Stack
- Python  
- Django  
- HTML, CSS

# How to Run
1. Clone the repository
   git clone https://github.com/priyabhansali25/Regulatory-Document-Management-System.git
2. Navigate to project folder
   cd regulatory_system
3. Install dependencies
   pip install -r requirements.txt 
4. Run server using
   python manage.py runserver
